# Video Meetings App with WebRTC

Create or join the room, send a link to friends, and enjoy a video meeting.

https://videoma.vercel.app/

## Tech Stack

- Frontend
  - React.js
  - WebRTC
  - Tailwindcss
- Backend
  - Node.js / Socket.io / Express.js / Mongoose
- Database
  - MongoDB
